Code Example of Chapter 8, in HTML5 Game Development
--------------------------------------------------------

01. making-connection: Establish the connection between client and server.
02. connection-count: Count how many connections connected to server.
03. send-message: Basic message sending.
04. basic-chat: Simple chat built on top of message sending.
05. local-drawing: Drawing approach by passing drawing data as messages.
06. sync-drawing: Sync the drawing between all connected clients.
07. draw-and-guess-game: Create the game flow for playing the draw-n-guess game.
08. decorating-the-game: Decorating the game with CSS.
