Code Example of Chapter 2, in HTML5 Game Development
--------------------------------------------------------

01. installing-jquery: Linking the jQuery library and ensure it is loaded.
02. placing-game-elements: Setup the game elements of Ping Pong.
03. getting-mouse-position: Changing CSS position of the DOM elements with jQuery.
04. moving-the-ball: Using game loop to move the ball and bounce away when hitting the edges.
05. collision-with-paddles: Basic collision detection between the ball and paddles in Ping-Pong game.
06. control-left-paddles: Automatically moving left paddles.
07. showing-scores: Dynamically changing HTML text and content with jQuery.
